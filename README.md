# JobFlow Scraper

Инструмент для автоматического сбора вакансий из Telegram-каналов и сохранения их в базу данных Supabase.

## Настройка базы данных
Для работы системы необходимо создать две таблицы в Supabase. Скопируйте и выполните следующий SQL-код в SQL Editor вашего проекта:

~~~sql
CREATE TABLE vacancies (
    id bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    title text NOT NULL,
    url text NOT NULL UNIQUE,
    description text,
    status text DEFAULT 'new',
    created_at timestamp with time zone DEFAULT timezone('utc'::text, now()) NOT NULL
);

CREATE TABLE active_channels (
    id bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    username text NOT NULL UNIQUE,
    last_message_id bigint DEFAULT 0,
    is_active boolean DEFAULT true,
    created_at timestamp with time zone DEFAULT timezone('utc'::text, now()) NOT NULL
);

INSERT INTO active_channels (username) VALUES ('sova_freelance');
~~~

## Установка и запуск

1. **Подготовка переменных окружения:**
   Создайте файл `.env` в корне проекта и добавьте в него ваши данные:
   ~~~env
   API_ID=ваш_api_id
   API_HASH=ваш_api_hash
   SUPABASE_URL=ваш_url_проекта
   SUPABASE_KEY=ваш_ключ_api
   ~~~

2. **Установка зависимостей:**
   ~~~bash
   pip install -r requirements.txt
   ~~~

3. **Запуск скрапера:**
   Для корректной работы импортов запускайте проект как модуль из корневой директории:
   ~~~bash
   python -m src.scraper
   ~~~
