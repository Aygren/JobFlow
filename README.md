# JobFlow

JobFlow — это инструмент для автоматизированного сбора вакансий из Telegram-каналов и их сохранения в базу данных SQLite. Проект находится в активной стадии разработки, базовая архитектура и каркас парсера уже реализованы.

## Возможности и статус
На текущий момент реализована структура БД и механизмы работы с ней. В процессе разработки находятся функции парсинга через Telegram API (Telethon) и система фильтрации вакансий.

## Технологический стек
Для работы используются Python 3.x, Telethon для взаимодействия с Telegram и SQLite3 для хранения данных.

## Установка и запуск
1. Клонируйте репозиторий и создайте виртуальное окружение: `python -m venv .venv`.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Создайте файл `.env` в корне проекта и добавьте в него ваши `API_ID` и `API_HASH`.
4. Запуск будет доступен после завершения реализации основного функционала парсинга.
